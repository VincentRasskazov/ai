# VincentAI

A fast, minimal AI chat interface built with pure HTML, CSS, and JavaScript. Supports multiple models, local WebGPU inference, Google login, and persistent chat history.

## Features
- Multi‑model support (DeepSeek, Llama, Qwen, GPT‑4o mini, and more)
- Local WebGPU inference for supported models
- Google Sign‑In for synced chat history
- Incognito mode for temporary sessions
- File uploads included in prompt context
- PWA support with offline capabilities

## Configuration
Update these constants in `index.html`:
- `BACKEND_URL`
- `GOOGLE_CLIENT_ID`

## Tech Stack
- HTML, CSS (custom properties), JavaScript
- Marked.js, Highlight.js, Font Awesome
- WebLLM for local browser inference
- LocalStorage for settings and history
